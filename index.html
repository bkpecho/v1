My Story

Hello! I'm Bryan, a full-stack software engineer specializing in web development. My coding journey began in 2013 when I stumbled upon the marquee element in HTML. Little did I know that this simple discovery would ignite a deep passion within me and set me on an extraordinary path towards a programming career.

Over the years, I've had the opportunity to delve into various programming languages. I started with Pascal and even took on freelancing projects using Visual Basic. During my college years, my focus shifted to Java and SQL. Today, my deep passion lies in JavaScript and the MERN stack (MongoDB, Express.js, React, and Node.js) for my development projects.

Lastly, as an advocate for open source initiatives, I invite you to connect with me if you share the same passion or have an exciting project in mind. Let's geek out!
